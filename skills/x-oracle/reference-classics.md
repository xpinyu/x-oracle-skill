# Classical Source Reference

Use this file to make X-Oracle reports feel grounded in traditional BaZi reasoning without fabricating quotations.

## Citation Rule
Do not invent verbatim classical quotes. If an exact quote is not verified, write:

- "按《子平真诠》重月令格局的思路..."
- "依《滴天髓》重气势流通、病药相对的原则..."
- "从《穷通宝鉴》的调候角度看..."
- "按子平十神六亲法..."

Name the principle, then show the chart evidence.

## Source Roles
### 《子平真诠》
Use for:

- Month command as the governing outline.
- Pattern formation and pattern failure.
- Clear vs mixed structures.
- Whether a god is usable, protected, damaged, or rescued.

Report language:

```markdown
按《子平真诠》重月令格局的思路，此盘先看[月令]。月令为[branch]，主气为[hidden stem]，天干又透出[stem/ten god]，所以不能只按五行数量断，而要看[pattern]能否成局。
```

### 《滴天髓》
Use for:

- Qi flow.
- Strength and weakness.
- Disease and medicine.
- Source and outlet.
- Whether the chart is blocked, flowing, cold, hot, dry, or wet.

Report language:

```markdown
依《滴天髓》重气势流通的原则，命局的关键不在某一颗星孤立好坏，而在[signal]是否能流向[signal]。此盘病在[problem]，药在[solution]。
```

### 《穷通宝鉴》
Use for:

- Seasonal climate.
- Cold/heat/dry/wet adjustment.
- Whether the day master needs warming, cooling, moistening, drying, or strengthening.

Report language:

```markdown
从《穷通宝鉴》的调候角度看，[day master]生于[month]，先论寒暖燥湿。若局中[climate problem]，则[useful element]比单纯扶抑更要紧。
```

### 《渊海子平》
Use for:

- Foundational ten-god meanings.
- Six relatives.
- Pillar-level interpretation.
- Basic combinations, clashes, and stem/branch relations.

Report language:

```markdown
按子平十神法，[ten god]代表[function]。此星在[location]透出/藏于[branch]，说明此事不是没有，而是以[visible/latent/blocked]的方式出现。
```

### 《三命通会》
Use for:

- Broad synthesis.
- Special patterns.
- Shen-sha as secondary confirmation.
- Complex mixed charts.

Report language:

```markdown
若参看《三命通会》的综合法，神煞只能作辅证，不能越过格局旺衰。此处[shen-sha/special signal]可以说明[secondary note]，但主判断仍以[main evidence]为准。
```

## Evidence Sentence Patterns
Use these sentence forms often:

- "此断的依据不是感觉，而是[chart evidence]。"
- "先看月令：[month branch]当令，故[conclusion]。"
- "[ten god]透干而[has root/no root]，所以现实中多表现为[manifestation]。"
- "[branch]冲[branch]，冲到[palace/domain]，应事多在[domain]。"
- "大运来[stem branch]，引动原局[signal]，所以这一阶段的主题是[theme]。"
- "若出生时辰有误，最先变化的是[hour pillar / timing / children / later-life reading]。"

## What Not to Do
- Do not cite a book to support a conclusion that has no chart evidence.
- Do not use shen-sha as the main reason when the basic pattern does not support it.
- Do not make fear-based claims.
- Do not force one reading if the chart has mixed signals.
